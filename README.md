# Data Structures - JS

A web-based interactive visualization of fundamental data structures implemented in JavaScript.

## Overview

This project provides a hands-on visual interface to explore and interact with three core data structures:

- **Stack** – Last In, First Out (LIFO)
- **Queue** – First In, First Out (FIFO)
- **Linked List** – Sequential node-based structure

## Features

- Interactive UI to push, pop, enqueue, dequeue, and manipulate nodes
- Real-time visual receipt/log of operations performed
- Clean and responsive design

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
